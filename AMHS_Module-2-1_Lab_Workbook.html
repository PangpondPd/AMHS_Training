chmod numeric quick reference:
  777 = rwxrwxrwx (everyone full — AVOID)
  755 = rwxr-xr-x (scripts, directories)
  644 = rw-r--r--  (regular data files)
  640 = rw-r-----  (sensitive config files)
  600 = rw-------  (private keys)
Rule: chmod gives NO output when it works.
If you see output, it is an error message.
